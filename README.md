我要做一個小說網頁
主色調：紅、藍、白

小說排列以出版日期新的在最前面，舊的在後面。

每一章的標題、作者、出版日期都會顯示在頁面上。

文章有2種,付費和免費。

付費的文章的邊框是紅色，免費的文章的邊框是藍色。

點擊免費文章標題可以打開文章內容。

點擊付費文章標題會顯示付費內容，需要先購買才能閱讀。

頁面上方一個搜索框，可以搜尋關鍵字,列出所有有關的文章。

==
